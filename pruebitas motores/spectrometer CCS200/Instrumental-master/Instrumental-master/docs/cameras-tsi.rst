Thorlabs TSI Cameras
====================

.. toctree::

This module is for controlling Thorlabs cameras that use the TSI SDK. Note that Thorlabs DCx cameras
use a separate SDK.


Installation
------------
This module requires the TSI SDK and the `NiceLib` package.
